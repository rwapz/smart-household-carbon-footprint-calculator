# Smart Household Carbon Footprint Calculator 🦈

A web-based application designed to help households track and reduce their carbon footprint. Developed as part of the Computer Science curriculum at **Sheffield Hallam University**.

## 🚀 Project Overview
This project provides a "Smart" interface for users to calculate environmental impact based on household data. The current build focuses on a high-fidelity Auth system (Login/Signup) following a **1100x700 px** design card layout.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Gradients), JavaScript (ES6)
- **Backend:** PHP (PDO for Database Connectivity)
- **Database:** MySQL (Work in Progress - CAPS naming convention)
- **Tools:** XAMPP, GitHub Desktop, Mockplus (Prototyping)

## 📁 File Structure
- `index.php`: Main entry point containing the Login and Signup forms.
- `style.css`: Full stylesheet including hover states and responsive-ready components.
- `script.js`: External JavaScript handling the interactive form toggling.
- `db.php`: Database connection script using PDO.

## ⚙️ Local Setup
1. Clone this repository into your XAMPP `htdocs` folder.
2. Ensure Apache and MySQL are running in the XAMPP Control Panel.
3. Access the project via: `http://localhost/task/smart-household-carbon-footprint-calculator/`
